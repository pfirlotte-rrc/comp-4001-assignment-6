# Portable automation script (apply_minikube_dockerenv.sh)

- Create an automation script to deploy the application with the name
“apply_minikube_dockerenv.sh”
- Force builds into Minikube's Docker daemon (Docker driver)
- Verify images inside the node's Docker
- Apply manifests
- Restart deployments to pick up local images
- Run the script manually and observe the pod creation.
- Launch the application with “minikube service nginx”
